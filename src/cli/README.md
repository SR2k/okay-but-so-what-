# SR2k CLI

One simple command, many files generated.

## Install

``` sh
yarn global add @sr2k/cli
```

or use NPM:

``` sh
npm i -g @sr2k/cli
```

## Usage

Firstly, init a NPM project:

``` sh
npm init
```

Then fire the `sr2k` command at project root:

``` sh
sr2k
```

Now, your `.editorconfig`, `.npmrc` and linters shall be ready.
